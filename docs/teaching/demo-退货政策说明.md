# 某某科技有限公司 退货与售后政策

> 版本：2025年1月修订版 | 生效日期：2025年1月1日

## 一、退货政策概述

本公司致力于为客户提供优质的产品和服务。如果您对购买的商品不满意，可以在规定时间内申请退货。本政策适用于通过本公司官方渠道（官网、直营门店、授权经销商）购买的所有商品。

## 二、退货时间要求

### 2.1 标准退货期限

自商品签收之日起 **15个自然日** 内，消费者可以无理由退货。超过15天但在30天内的，需要提供商品质量问题的相关证明。

### 2.2 特殊商品退货期限

- **电子产品**：7个自然日内（自签收日起算）
- **食品类商品**：不支持无理由退货，仅限质量问题
- **定制商品**：不支持退货，除非存在严重质量缺陷
- **大促活动商品**：按活动页面标注的退货政策执行

## 三、退货条件

### 3.1 无理由退货条件（15天内）

满足以下 **全部条件** 方可无理由退货：

1. 商品未经使用、未拆封、未损坏
2. 商品原包装完好，配件齐全（含说明书、保修卡、赠品等）
3. 提供有效的购买凭证（订单号、发票或收据）
4. 通过本公司指定的退货申请渠道提交申请

### 3.2 质量问题退货条件（30天内）

商品存在以下质量问题的，可在30天内申请退货：

1. 商品存在功能性故障，无法正常使用
2. 商品与描述严重不符
3. 商品存在安全隐患
4. 收到商品时已损坏

质量问题退货需提供：照片或视频证据、商品故障描述、购买凭证。

## 四、退货流程

### 步骤一：提交退货申请

通过以下任一渠道提交：
- **官网**：登录账户 → 我的订单 → 申请退货
- **客服热线**：400-XXX-XXXX（工作日 9:00-18:00）
- **微信公众号**：关注"某某科技"→ 在线客服

### 步骤二：等待审核

客服人员将在 **1-3个工作日** 内审核退货申请，审核通过后发送退货地址和退货单号。

### 步骤三：寄回商品

收到退货确认后，请在 **7个工作日** 内将商品寄回指定地址。请妥善包装商品，避免运输损坏。

### 步骤四：退款处理

收到退回商品并验收通过后，将在 **5-10个工作日** 内完成退款。退款将原路返回至支付账户。

## 五、退货运费

### 5.1 运费承担规则

| 退货原因 | 运费承担方 |
|---------|-----------|
| 商品质量问题 | 本公司承担 |
| 消费者无理由退货 | 消费者自行承担 |
| 发错商品 | 本公司承担 |
| 商品与描述不符 | 本公司承担 |

### 5.2 运费标准

本公司承担运费的情况下，消费者可选择以下方式：
- 本公司安排上门取件（推荐）
- 消费者自行寄回，凭快递单据报销（上限30元）

## 六、不予退货的情况

以下情况不接受退货申请：

1. 超过退货期限
2. 商品已使用、损坏或包装不完整
3. 无法提供有效购买凭证
4. 人为损坏、改装或私自维修过的商品
5. 消耗品（如墨盒、电池等已拆封使用的）
6. 赠品已使用但主商品要求退货的

## 七、换货政策

### 7.1 换货条件

商品存在质量问题且消费者希望换货而非退货的，可在30天内申请同款换货。如同款商品无库存，可协商更换同价位其他商品或退货退款。

### 7.2 换货流程

换货流程与退货流程相同，在提交申请时选择"换货"选项即可。换货产生的双向运费均由本公司承担。

## 八、售后保修

### 8.1 保修期限

- **电子产品**：自购买之日起1年
- **配件产品**：自购买之日起6个月
- **软件产品**：自购买之日起3个月

### 8.2 保修范围

保修期内因产品本身质量问题导致的故障，本公司提供免费维修或更换服务。以下情况不在保修范围内：
- 人为损坏（跌落、进水、拆机等）
- 自然灾害造成的损坏
- 未经授权的第三方维修
